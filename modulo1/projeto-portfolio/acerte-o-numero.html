Projeto Acerte o Número

Este projeto é um jogo contra o computador. Nele, você é desafiado a acertar um número entre 1 e 100 que foi escolhido aleatoriamente pelo computador.
O objetivo é acertar o número com a menor quantidade de chutes.

  1. Primeiro, você digita o número
  2. No console, você acompanha as suas tentativas e recebe dicas que ajudam a acertar o número
  3. Continue tentando até finalmente acertar!
